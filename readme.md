# Kanban Board

A simple and interactive Kanban Board built using HTML, CSS, and JavaScript.
Users can create tasks, drag them between different columns, and all data is saved automatically using localStorage.
This project focuses on clean UI, smooth drag-and-drop, and basic task management.

## Features

- Add New Tasks
- Click Add new Task to open a modal
- Enter Title and Description
- The task automatically appears under To-Do

## Drag & Drop

Move tasks freely between:

- To Do
- In Progress
- Done
- Smooth hover animation when dragging over columns

## Local Storage Persistence

- Tasks remain saved even after refreshing the page
- Page reloads with the exact same task positions

## Automatic Task Counters

Each column displays a live count of the number of tasks it contains.

This updates whenever you:

- Create a task
- Drag a task
- Move a task to another column

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage API
